# Algorithms_and_DataStructure
アルゴリズムとデータ構造の本のコードをここに書いていく。


## ビルド
```docker-compose up -d```

## bash
```docker container exec -it cppdev /bin/sh```    

## プログラム実行
```cd /cppdev/src```

```g++ hello.cpp -o hello```

```./hello```
